### 🤔 Como rodar o projeto? 

```bash
# Clone o repositório
git clone https://github.com/Gabriek0/portfolio-2.0.git

# Mude para o diretório do projeto
cd https://github.com/Gabriek0/portfolio-2.0.git

# Instale as dependências
yarn

# Rode o projeto
yarn dev
```
